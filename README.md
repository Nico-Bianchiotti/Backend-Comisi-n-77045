# Plataforma de Eventos e Inscripciones

## Descripción

API REST para gestionar eventos e inscripciones.

## Tecnologías

- Node.js
- Express
- dotenv

## Instalación

npm install

## Variables de entorno

Crear un archivo .env con:

PORT=
NODE_ENV=
MONGO_URL=
JWT_SECRET=

## Ejecutar

npm run dev

## Estructura

config/
routes/
controllers/
services/
repositories/
dao/
models/
middlewares/
utils/

## Rutas

GET /api/health

GET /api/events

GET /api/sessions
